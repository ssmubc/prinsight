# PRInsight

ML-powered GitHub PR intelligence agent

## What it does
- Analyzes team PR patterns
- Predicts merge times with ML
- Suggests optimal reviewers
- Identifies bottlenecks

## Tech Stack
- TypeScript MCP server
- Python XGBoost ML model
- Claude API for agent reasoning
- Next.js demo UI
